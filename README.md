## 📖 Overview

The **Self-Driving Cars Specialization** by the **University of Toronto** provides a practical and theoretical introduction to autonomous vehicle engineering. Throughout the four-course series, you'll learn how modern self-driving cars perceive the environment, estimate their position, plan safe trajectories, and control vehicle motion.

The specialization combines lectures with hands-on programming assignments using the **CARLA simulator**, allowing students to implement key autonomous driving algorithms in realistic driving scenarios. :contentReference[oaicite:0]{index=0}

## 📚 Courses

### 🚘 Course 1 — Introduction to Self-Driving Cars

Learn the foundations of autonomous driving systems.

### Topics

- History of Autonomous Vehicles
- SAE Levels of Automation
- Autonomous Vehicle Software Stack
- Vehicle Dynamics
- Longitudinal & Lateral Control
- Sensors (Camera, LiDAR, Radar, GPS, IMU)
- Operational Design Domain (ODD)
- Object and Event Detection and Response (OEDR)
- Safety Engineering
- CARLA Simulator

### Final Project

Implement a vehicle controller capable of navigating a racetrack in the **CARLA** simulator. :contentReference[oaicite:1]{index=1}

---

### 📍 Course 2 — State Estimation and Localization for Self-Driving Cars

Learn how autonomous vehicles determine **where they are**.

### Topics

- Coordinate Frames
- GPS
- IMU
- LiDAR Localization
- Dead Reckoning
- Kalman Filters
- Extended Kalman Filter (EKF)
- Particle Filters
- Sensor Fusion
- Visual Odometry
- Simultaneous Localization and Mapping (SLAM)

### Final Project

Develop localization algorithms by combining multiple sensor measurements. :contentReference[oaicite:2]{index=2}

---

### 👁️ Course 3 — Visual Perception for Self-Driving Cars

Focus on how autonomous vehicles understand their surroundings using computer vision.

### Topics

- Camera Geometry
- Image Formation
- Feature Detection
- Feature Matching
- Object Detection
- Object Tracking
- Semantic Segmentation
- Convolutional Neural Networks (CNNs)
- Deep Learning
- Visual Perception Pipeline

### Final Project

Implement perception algorithms for detecting and tracking objects within the CARLA simulation environment. :contentReference[oaicite:3]{index=3}

---

### 🛣️ Course 4 — Motion Planning for Self-Driving Cars

Build the decision-making and planning system of an autonomous vehicle.

### Topics

- Route Planning
- Behavior Planning
- Motion Planning
- Trajectory Generation
- Dijkstra Algorithm
- A* Search
- Finite State Machines
- Occupancy Grid Maps
- Collision Checking
- Velocity Planning
- Obstacle Avoidance

### Final Project

Implement a complete hierarchical motion planner capable of:

- Following lanes
- Avoiding parked vehicles
- Following traffic
- Navigating intersections
- Safely avoiding collisions

using the **CARLA simulator**. :contentReference[oaicite:4]{index=4}
